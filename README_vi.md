# 🎬 T Creator v1.0.0 - Tạo ảnh & video AI [![English](https://img.shields.io/badge/English-blue)](README.md) [![中文](https://img.shields.io/badge/中文-red)](README_zh.md)

**T Creator** là ứng dụng desktop tạo ảnh và video AI bằng các model lớn với giá rất rẻ. Tạo ảnh với **Nano Banana**, video với **Veo 3**, và các model khác (Grok, Sora, Seedance 2.0) sắp ra mắt. Hỗ trợ **Windows** và **Mac**; bản web coming soon.

-----

## ✨ Tính năng chính

* **🖼️ Tạo ảnh AI:** Văn bản thành ảnh và thành phần thành ảnh (nhiều ảnh tham chiếu). Model: **Nano Banana Pro**.
* **🎬 Tạo video AI:** Văn bản thành video, khung hình thành video (ảnh + chuyển động), thành phần thành video. Model: **Veo 3** (Fast / Quality).
* **⚙️ Điều khiển đầy đủ:** Chọn model, chất lượng, tỷ lệ khung hình; với video: thời lượng. Prompt hàng loạt (mỗi dòng một prompt) với Start / Pause / Stop.
* **📊 Tiến trình thời gian thực:** Xem trạng thái và tiến độ từng prompt; xem trước và mở kết quả trong app.
* **📂 Tự động tải xuống:** Lưu ảnh và video vào thư mục bạn chọn (tùy chọn).
* **📜 Lịch sử:** Duyệt và xem trước các lần tạo trước.
* **🌐 Đa ngôn ngữ:** English, Tiếng Việt, 中文.

-----

## 📥 Cài đặt

1. Vào [Releases](https://github.com/trgkyle/t-creator-user-guide/releases).
2. Tải bản cài đặt theo hệ điều hành:
   - **Windows:** file `.exe` (vd. `T Creator Setup 1.0.0.exe`).
   - **Mac:** file `.dmg` (vd. `T Creator-1.0.0-arm64.dmg`).
3. Chạy file cài đặt và mở **T Creator**.

## 📖 Hướng dẫn sử dụng

### Bắt đầu

1. **Đăng ký / Đăng nhập**  
   Tạo tài khoản hoặc đăng nhập. Credits dùng để trừ cho mỗi lần tạo.

2. **Mở Scene Creator**  
   Màn hình chính là **Scene Creator**: tạo ảnh hoặc video từ prompt.

3. **Chọn tab**  
   - **Tạo ảnh** — văn bản thành ảnh hoặc thành phần thành ảnh.  
   - **Tạo video** — văn bản thành video, khung hình thành video, hoặc thành phần thành video.

4. **Thiết lập**  
   - **Chế độ:** VD Text to Image, Components to Image, Text to Video, Frame to Video.  
   - **Model:** VD Nano Banana Pro (ảnh), Veo3 Fast / Veo3 Quality (video).  
   - **Chất lượng**, **Tỷ lệ khung hình**; với video thêm **Thời lượng**.

### 1. Ảnh: Văn bản thành ảnh

1. Tab **Tạo ảnh**, chế độ **Text to Image**.
2. Nhập một hoặc nhiều prompt (mỗi dòng một prompt; dòng trống = prompt mới).
3. Nhấn **Start** để chạy. Xem trạng thái và preview trong bảng bên phải.

### 2. Ảnh: Thành phần thành ảnh

1. Tab **Tạo ảnh**, chế độ **Components to Image**.
2. Tải lên 1–5 ảnh tham chiếu (kéo thả hoặc chọn file).
3. Nhập prompt (mỗi dòng một prompt). Mỗi prompt dùng chung bộ ảnh đã tải.
4. Nhấn **Start**.

### 3. Video: Văn bản thành video

1. Tab **Tạo video**, chế độ **Text to Video**.
2. Nhập prompt (mỗi dòng một prompt).
3. Nhấn **Start**.

### 4. Video: Khung hình thành video

1. Tab **Tạo video**, chế độ **Frame to Video**.
2. Tải lên 1–2 ảnh nguồn.
3. Nhập prompt (mỗi dòng một prompt). Mỗi prompt áp dụng cho ảnh đã tải.
4. Nhấn **Start**.

### 5. Video: Thành phần thành video

1. Tab **Tạo video**, chế độ **Ingredients to Video**.
2. Tải lên tối đa 3 ảnh tham chiếu.
3. Nhập prompt và nhấn **Start**.

### Kết quả và tải xuống

- **Kết quả:** Mỗi dòng hiển thị prompt, trạng thái (pending / running / done / failed) và preview. Nhấn vào preview để xem full size.
- **Tự động tải xuống:** Bật **Auto Download Image** và/hoặc **Auto Download Video** ở footer và chọn thư mục. File hoàn thành sẽ được lưu tự động.

## 💡 Mẹo

1. **Prompt:** Viết rõ ràng; prompt cụ thể thường cho kết quả tốt hơn.  
2. **Hàng loạt:** Mỗi dòng một prompt; dòng trống để tách prompt.  
3. **Pause / Stop:** Pause để tạm dừng hàng đợi, Stop để hủy phần còn lại.  
4. **Credits:** Kiểm tra số dư; nạp thêm khi sắp hết.
